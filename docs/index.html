Features

🎯 Zod Schemas

Every parameter has a Zod type with .describe() hints. Works with AI structured output, form validation, or plain TypeScript. Zod 3 + 4 compatible.

🤖 AI URL Generation

One-function generateUrl() — give it a portal, intention, and natural language story. Uses Vercel AI SDK under the hood.

🔗 Custom Serializers

Each portal has its own URL serializer. Query params, JSON blobs, path DSLs, compound params — we handle the encoding.

📍 Location Codes

Verified neighborhood and city codes. 39 Oslo districts, 115+ NYC neighborhoods (Zillow + StreetEasy), 45 SF neighborhoods, and more.

🔄 Normalized Taxonomy

Adapters convert one set of search params across all portals. Write once, search everywhere.

📋 JSON-LD Extraction

Extract and query schema.org structured data from listing pages. Lightweight, zero dependencies beyond regex + JSON.parse.

AI-Powered URL Generation

Natural language in, parameterized search URL out. Works with any AI SDK provider.

generateUrl 
import { generateUrl } from "@use_homi/real-estate-portal-schemas/generate";

const result = await generateUrl({
  portal: "finn.no",
  intention: "rent",
  story: "Young couple in Oslo with a dog. 2BR, balcony, elevator. 15k NOK/month.",
});

if (result.ok) {
  console.log(result.url);    // https://www.finn.no/realestate/lettings/search.html?location=0.20061&...
  console.log(result.label);  // "Finn.no — Oslo Rent 2BR"
  console.log(result.params); // { location: "0.20061", min_bedrooms: 2, ... }
}

Low-Level: Direct Schema Usage

Full control with Zod schemas and custom serializers.

serialize 
import { finnNoConfig } from "@use_homi/real-estate-portal-schemas";

// Build a rental search URL for Oslo
const url = finnNoConfig.serialize(finnNoConfig.baseUrls.rent!, {
  location: "0.20061",      // Oslo
  price_to: 15000,           // max 15,000 NOK/month
  min_bedrooms: 2,
  facilities: ["1", "4"],    // balcony, elevator
  animals_allowed: "1",      // pets OK
});

// → https://www.finn.no/realestate/lettings/search.html?location=0.20061&price_to=15000&min_bedrooms=2&facilities=1&facilities=4&animals_allowed=1
